QR Code Generator (Python)

A simple Python project that generates a QR code for any website URL and saves it as an image file. The program takes a URL and a desired file name from the user, generates the QR code, and stores it in PNG format.

Features

Generate QR codes for any website URL

Automatically saves the QR code as a .png image

Adds the .png extension automatically if the user does not provide it

Simple and beginner-friendly Python implementation

Requirements

Python 3.x

qrcode library

Install the required library using:

pip install qrcode[pil]

Tech Stack

Python

qrcode library

Purpose

This project is a beginner-friendly Python utility that demonstrates how to generate QR codes programmatically and handle simple user input and file operations.
